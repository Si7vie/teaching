Week1 国民收入核算与国际收支

核心概念

GDP：国内生产总值，强调地域范围，是一国境内所有生产活动的最终成果。

GNP：国民生产总值，强调生产要素所有权，是一国居民在国内外的生产活动成果。

开放经济恒等式：Y = C + I + G + CA，其中CA为经常账户余额。

CA = EX - IM，代表一国与外国之间的净贸易状况。

国际收支账户

经常账户：记录货物、服务、收入和经常转移。

资本账户：记录资本转移和非生产、非金融资产的交易。

金融账户：记录金融资产和负债的交易。

复式记账原则：每笔交易都有借贷双方，总额相等。

重要公式

GDP = C + I + G + (EX - IM)

GNP = GDP + 本国居民从外国获得的收入 - 外国居民从本国获得的收入

经常账户余额 = 货物贸易余额 + 服务贸易余额 + 收入余额 + 经常转移余额

关键术语

经常账户：反映一国与外国之间的实际资源转移。

资本账户：记录资本转移和非生产资产的交易。

金融账户：记录金融资产和负债的跨境交易。

国际收支平衡表：系统记录一国与外国之间所有经济交易的报表。

简答题练习

1. 简述汇率超调的含义与原因。

含义：货币供给变化后，汇率先出现过度波动，再逐步回调到长期均衡水平。
原因：短期价格具有黏性，无法及时调整，所有压力由利率和汇率承担；长期价格逐步调整，汇率回归长期均衡。

2. 简述无抛补利率平价的经济含义。

当外汇市场均衡时，两国货币存款的预期收益率相等。如果不相等，资金会流向收益更高的货币，推动汇率变动，直到收益相等。
