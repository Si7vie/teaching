一价定律

同质商品在全球各地的价格，剔除汇率因素后应该相同。

一价定律成立的条件：商品同质、无交易成本、无贸易壁垒。

一价定律是购买力平价的基础。

购买力平价 PPP

绝对PPP：E = P/Pf，其中E为汇率，P为本国价格水平，Pf为外国价格水平。

相对PPP：ΔE/E = ΔP/P - ΔPf/Pf，汇率变化率等于两国通胀率之差。

PPP在长期比短期更接近现实。

费雪效应

费雪效应：名义利率 = 实际利率 + 预期通胀率。

国际费雪效应：两国名义利率之差等于预期通胀率之差。

通胀↑ → 名义利率↑ → 本币贬值（根据PPP）。

重要公式

绝对PPP：E = P/Pf

相对PPP：ΔE/E = π - πf，其中π为本国通胀率，πf为外国通胀率。

费雪效应：i = r + πe，其中i为名义利率，r为实际利率，πe为预期通胀率。

国际费雪效应：i - if = πe - πef

简答题练习

1. 简述汇率超调的含义与原因。

含义：货币供给变化后，汇率先出现过度波动，再逐步回调到长期均衡水平。
原因：短期价格具有黏性，无法及时调整，所有压力由利率和汇率承担；长期价格逐步调整，汇率回归长期均衡。

2. 简述无抛补利率平价的经济含义。

当外汇市场均衡时，两国货币存款的预期收益率相等。如果不相等，资金会流向收益更高的货币，推动汇率变动，直到收益相等。
